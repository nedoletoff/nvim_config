# Новые плагины для AstroNvim

Документация по недавно добавленным плагинам в конфиг.

## 1. Bad Apple (bad-apple.lua)

**Описание:** Проигрывает Bad Apple анимацию в Neovim.

**Использование:**
```
:BadApple
```

**Клавиша быстрого вызова:**
```
<leader>ba
```

**Требования:** libcanberra (опционально для звука)

---

## 2. Счётчик введённых символов (statusline-custom.lua)

**Описание:** Счётчик количества введённых символов за сессию, отображается на нижней панели.

**Как это работает:**
- Инициализируется при запуске Neovim
- Считает символы, введённые в режиме Insert (InsertCharPre)
- Показывает значок 󰌌 + число в statusline

**Использование:**
Автоматически включено, видно в statusline справа.

**Пример:**
```
 󰌌 Keys: 2345
```

---

## 3. Doom-like UI (doom-ui.lua)

Добавлены два плагина для красивого UI, похожего на Doom Emacs:

### 3.1 which-key.nvim

**Описание:** Показывает доступные клавиши и их привязки при нажатии leader.

**Использование:**
```
<leader>
```

После нажатия вы увидите красивое меню со всеми доступными командами.

**Настройки:**
- Preset: "helix"
- Иконки для навигации
- Timeout: 300ms

### 3.2 noice.nvim

**Описание:** Красивое отображение сообщений, поиска, и command palette.

**Особенности:**
- Красивая command palette внизу экрана
- Улучшенное отображение LSP документации
- Красивые сообщения вместо обычных
- Better search UI

**Требования:**
```
MunifTanjim/nui.nvim
```

---

## Установка

Все плагины добавлены в `lua/plugins/` и автоматически подгружаются lazy.nvim при запуске Neovim.

### Если требуются зависимости:

```bash
# Для Bad Apple с звуком (опционально)
sudo apt install libcanberra0

# Для других плагинов зависимостей нет (они управляются lazy.nvim)
```

---

## Файлы

- `lua/plugins/bad-apple.lua` - Bad Apple плагин
- `lua/plugins/statusline-custom.lua` - Счётчик символов
- `lua/plugins/doom-ui.lua` - Which-key и Noice плагины

---

## Удаление плагина

Если нужно отключить какой-то плагин, закомментируй его в файле или удали файл и запусти `:Lazy sync` в Neovim.
